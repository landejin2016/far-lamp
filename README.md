主要技术：Vue/vue-router/axios/vuex

## git commit规范
```
<type>(<scope>): <subject>
// 空行
<body>
// 空行
<footer>
```

 type：用于说明commit的类型，可选值如下：

- feat: 新增
- fix: 修复bug
- docs: 仅仅修改了文档
- style: 仅仅修改了样式
- refactor: 代码重构
- perf: 优化：性能、用户体验等
- test: 新增、修改测试用咧等
- chore: 改变构建流程或增加依赖库、工具等
- revert: 版本回滚

scope：commit影响的范围，如pages、components等
subject：commit描述

body：提交的具体说明
footer：备注
